# Instruções para Adicionar Ícone do Aplicativo

## Arquivos Necessários

Para que o ícone apareça corretamente em todos os lugares, você precisa criar os seguintes arquivos na raiz do seu servidor:

1. **icon-192.png** - Ícone 192x192 pixels (PNG)
2. **icon-512.png** - Ícone 512x512 pixels (PNG)

## Onde o Ícone Aparecerá

✅ **Aba do navegador** (favicon)
✅ **Aplicativo desktop** (Chrome/Edge)
✅ **Aplicativo web mobile iOS** (quando adicionado à tela de início)
✅ **Tela de inicialização** do aplicativo

## Como Criar os Ícones

1. Crie uma imagem quadrada (recomendado: 512x512 pixels)
2. Use um editor de imagens (Photoshop, GIMP, Canva, etc.)
3. Exporte em dois tamanhos:
   - 192x192 pixels → salve como `icon-192.png`
   - 512x512 pixels → salve como `icon-512.png`
4. Coloque os arquivos na mesma pasta do `index.html`

## Dicas

- Use fundo transparente ou sólido
- Mantenha o design simples e reconhecível em tamanhos pequenos
- Teste como o ícone aparece em diferentes tamanhos
- Use cores contrastantes para melhor visibilidade

## Verificação

Após adicionar os arquivos:
1. Recarregue a página (Ctrl+F5 ou Cmd+Shift+R)
2. Verifique se o ícone aparece na aba do navegador
3. Teste a instalação do aplicativo desktop
4. No iOS, teste "Adicionar à Tela de Início"
